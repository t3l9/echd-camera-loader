## Описание

Приложение предназначено для массовой выгрузки фотографий с камер ЕЦХД по заданным параметрам: районы, доля данных, очередь обработки. Поддерживается работа с Google Sheets для хранения и управления данными.

## Возможности

- Автоматическая авторизация на портале ЕЦХД
- Выгрузка камер по выбранным районам
- Фильтрация по доле данных (20%, 50%, 100%)
- Выбор очереди обработки (1-5 очередь)
- Случайный или последовательный порядок обработки
- Отложенный запуск по расписанию
- Автоматическое переименование файлов по адресам
- Сохранение отчёта о проблемных камерах
- Проверка доступных обновлений

## Требования

- Python 3.11 или выше
- Google Chrome
- Учётная запись ЕЦХД
- Доступ к Google Sheets (опционально)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd echd-camera-loader
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:

Windows:
```bash
venv\Scripts\activate
```

Linux/Mac:
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Настройте переменные окружения:
```bash
# Скопируйте шаблон
copy .env.example .env

# Отредактируйте .env файл, вставив свои данные:
# - GOOGLE_SHEETS_URL - URL вашей Google таблицы
# - GOOGLE_SERVICE_ACCOUNT_PRIVATE_KEY - приватный ключ сервисного аккаунта
# - Остальные параметры Google Service Account
```

## Запуск

```bash
python main.py
```

## Использование

### Главная вкладка

1. Нажмите "Загрузить листы" для получения списка из Google Sheets
2. Выберите нужный лист из списка
3. Нажмите "Далее"

### Окно настройки

1. Введите логин и пароль от ЕЦХД
2. Выберите районы для обработки
3. Настройте параметры:
   - Доля данных (20%, 50%, 100%)
   - Очередь обработки
   - Случайный порядок (опционально)
4. Нажмите "Запустить обработку"

### Отложенный запуск

1. Перейдите на вкладку "Отложенный запуск"
2. Выберите дату и время запуска
3. Программа автоматически начнёт работу в указанное время

### Обновления

1. Перейдите на вкладку "Обновления"
2. Нажмите "Проверить обновления"
3. При наличии новой версии следуйте инструкциям

## Результаты работы

После завершения обработки в папке загрузок создаётся папка с именем листа и датой. Структура:

```
Downloads/
└── <Имя_листа>_<дата>/
    ├── <Район 1>/
    │   ├── <адрес_камеры_1>.jpg
    │   └── ...
    ├── <Район 2>/
    │   └── ...
    └── ПРОБЛЕМНЫЕ_КАМЕРЫ.xlsx
```

Файл `ПРОБЛЕМНЫЕ_КАМЕРЫ.xlsx` содержит:
- Тип проблемы (без положения / не выгружена)
- Район
- Адрес
- Номер камеры
- Положение
- Описание ошибки

## Настройка Google Sheets

Приложение использует сервисный аккаунт для доступа к Google Sheets. Ключи уже встроены в конфигурацию.

Требуемые разрешения для листов:
- `Район` — название района
- `Номер камеры` — идентификатор камеры
- `Адрес` — адрес установки (для переименования файлов)
- `Положение` — положение камеры (опционально)

## Известные ограничения

- Требуется стабильное интернет-соединение
- Для работы необходим Google Chrome
- Некоторые камеры могут не выгружаться из-за проблем на стороне ЕЦХД
- При отложенном запуске компьютер не должен уходить в спящий режим

## Рекомендации

1. Перед запуском убедитесь, что браузер Chrome закрыт
2. Для отложенного запуска отключите спящий режим:
   - Win+I → Система → Питание и спящий режим → "Никогда"
3. Регулярно проверяйте наличие обновлений
4. Сохраняйте отчёты о проблемных камерах для последующей обработки
